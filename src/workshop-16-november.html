  • Description

    In this workshop you will learn how to create, build and deploy a very simple website from scratch. I will cover some basic background information about each technology and tool we will use as well as show how to actually put it into practice.

    The tools and technologies we will be using in this workshop:

    • HTML/HTML5 (HyperText Markup Language)

    • CSS/CSS3 (Cascading Style Sheets)

    • Grunt

    • Amazon S3 - they have a free tier so you won't need to pay for any hosting, but you will need billing details

    • Sublime or your preferred text editor

    • Terminal like iTerm Terminal or your preferred one

    By the end of the workshop you will have a live simple website that anyone can access on the internet.

  • What will you need?

    • Laptop (Unix/Linux or Mac, if you are on Windows please contact me first and if there is enough demand generate relevant instructions)

    • Basic command line skills preferable - if you have never used a terminal before I can provide some basic examples and things to practice before class.

    • Make sure your laptop can connect to the wifi as we will need to install all the tools during workshop, WiFi will be available for guest access.

    • Credit Card for Amazon signup, no payment is needed but to create an account they ask for details.

  • Topics and tasks we will cover during the workshop

    • What is HTML?

    • Some HTML basics (everything you need to know to create a simple page)

    • What is CSS?

    • Some basic CSS properties

    • Create a CSS Stylesheet to style the HTML page

    • How to structure your projects (folder, files etc)

    • How to build you project using Grunt and do some other cool stuff too

    • Creating a free account on Amazon S3

    • Creating a bucket on S3 to host our website

    • Creating all the relevant users/permissions on S3
